How to Install Docker & Set Up gem5 Simulator

A step-by-step guide for beginners

Introduction

This guide will teach you how to install Docker and set up the gem5 simulator within a Docker container. It is designed for complete beginners and includes detailed steps and images to make the process easy to follow.

Prerequisites

No prior knowledge of Docker or gem5 is required. Ensure that your computer has at least 4GB of RAM and enough storage for Docker images.

Step 1: Installing Docker

Windows

  1. Download Docker Desktop from Docker's official site.

  2. Run the installer and select "Use WSL 2". Follow the installation instructions, then restart your computer if prompted.

  3. Open Docker Desktop and wait for it to start. To verify installation, open Command Prompt and run:

    docker --version

macOS

  1. Download Docker Desktop from Docker's official site.

  2. Drag the Docker icon to the Applications folder, then launch Docker from Applications.

  3. Open Terminal and verify installation by running:

    docker --version

Ubuntu

  1. Update system packages by running:

    sudo apt update

  2. Install Docker with:

    sudo apt install docker.io -y

  3. Start Docker and enable it to run at boot:

    sudo systemctl start docker
    sudo systemctl enable docker

  4. Verify installation:

    docker --version

  5. Add your user to the Docker group to use Docker without 'sudo':

    sudo usermod -aG docker $USER

  6. Log out and back in for changes to take effect.

Step 2: Setting Up gem5 in Docker

  1. Create a directory for gem5 on your host machine:

    mkdir ~/gem5

  2. Pull and run the gem5 Docker image with:

    docker run -tid --hostname=gem5_dev --name=gem5_container -v ~/gem5:/root ghcr.io/gem5/ubuntu-24.04_all-dependencies:v24-0 /bin/bash

  3. Access the Docker container:

    docker exec -ti gem5_container /bin/bash

  4. Clone and build gem5 inside the container:

    git clone https://github.com/gem5/gem5.git
    cd gem5
    scons build/X86/gem5.opt -j8

  5. Run a basic gem5 simulation to verify the setup:

    build/X86/gem5.opt configs/learning_gem5/part1/simple.py
Note: Building gem5 may take some time, depending on your computer's performance.

Conclusion

You have successfully installed Docker, set up a containerized environment, and configured gem5 within Docker. This setup provides a consistent and isolated environment for exploring gem5 simulations.
